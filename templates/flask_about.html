About This Flask Example

This simple Flask application demonstrates the following concepts:

  • Routing: Mapping URLs to functions
  • Templates: Using Jinja2 for HTML templates
  • Template Inheritance: Reusing common layout elements
  • Form Handling: Processing user input
  • Dynamic URLs: Using variable rules in routes

Flask Basics

Flask is a micro web framework written in Python. It's designed to make getting started quick and easy, with the ability to scale up to complex applications.

Key Features Demonstrated:

  • @app.route() decorator for defining routes
  • render_template() for rendering HTML templates
  • request object for handling form data
  • redirect() and url_for() for navigation
  • Template inheritance with {% extends %} and {% block %}

How to Run This Example

  1. Save the Python code in a file (e.g., app.py)
  2. Install Flask: pip install flask
  3. Run the application: python app.py
  4. Open your browser to http://127.0.0.1:5000
