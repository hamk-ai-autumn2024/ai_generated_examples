How to Install Flask

A step-by-step guide to setting up Flask for Python web development

What is Flask?

Flask is a lightweight web framework for Python that provides essential tools and features for building web applications. It's designed to make getting started quick and easy, with the ability to scale up to complex applications.

Prerequisites

1

Install Python

Flask requires Python 3.7 or newer. If you don't have Python installed:

  1. Visit python.org
  2. Download the latest version for your operating system
  3. Run the installer and make sure to check "Add Python to PATH" during installation
2

Verify Python Installation

Open your terminal/command prompt and run:

python --version

You should see output like Python 3.x.x. If you get an error, Python may not be installed correctly or not added to your PATH.

Installing Flask

3

Using pip (Recommended)

The easiest way to install Flask is using pip, Python's package installer:

pip install Flask

If you have both Python 2 and 3 installed, you might need to use:

pip3 install Flask
4

Using Virtual Environments (Highly Recommended)

Virtual environments isolate your project dependencies. This is considered a best practice:

Create a Virtual Environment

# Create a virtual environment named "venv" python -m venv venv # Activate the virtual environment # On Windows: venv\Scripts\activate # On macOS/Linux: source venv/bin/activate

Install Flask in the Virtual Environment

pip install Flask

Deactivate the Virtual Environment

deactivate

Verifying Installation

5

Check Flask Version

After installation, verify Flask is installed correctly:

python -c "import flask; print(flask.__version__)"

You should see the Flask version number.

6

Create a Simple Test Application

Create a file named app.py with the following content:

from flask import Flask app = Flask(__name__) @app.route('/') def hello(): return 'Hello, Flask!' if __name__ == '__main__': app.run(debug=True)

Run the application:

python app.py

You should see output similar to:

* Running on http://127.0.0.1:5000 * Debug mode: on

Open your web browser and go to http://127.0.0.1:5000 to see your application.

Troubleshooting

Note: If you encounter permission errors when installing Flask, try using the --user flag:

pip install --user Flask

Warning: On some systems, you might need to use python3 and pip3 instead of python and pip if you have both Python 2 and 3 installed.
