About This Flask Demo

This simple Flask application demonstrates the following concepts:

Flask Basics

Flask is a micro web framework written in Python. It's designed to make getting started quick and easy, with the ability to scale up to complex applications.

Key Features:

  • Built-in development server and debugger
  • Integrated support for unit testing
  • RESTful request dispatching
  • Jinja2 templating
  • Secure cookies support (client side sessions)
  • 100% WSGI 1.0 compliant
